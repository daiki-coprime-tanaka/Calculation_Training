# Calculation_Training
CLI Calculation Training with output files

# files
- Calculaton.c

 Output 5 kind of calculation(add, subtract, multiply, gauss division, reminder) 
- CState.txt

 Memory serial number of output file of Calculation.c
- Multiply.c

 Output only multiply calculation
- State.txt

 Memory serial number of output file of Multiply.c

# outputs
- No.xx-c.txt(Calculation.c)/No.xx.txt(Multiply.c)

 To view result personally.
- No.xx-cf.txt(Calculation.c)/No.xxf.txt(Multiply.c)

 To convert it to pdf via CalcMultiplyRes.xlsx.

# How to convert formated text file to PDF?
1. Open formatted text file from Excel. Select ファイル>開く>このPC(Forced translaion : Files>Open>This PC).
2. Select コンマやタブなどの区切り文字によってフィールドごとに区切られたデータ(Forced translation : Data separated by commas, tabs, or so on into field) in text wizard.
3. Select その他(Other) and type "|" in textbox.
4. Press 完了(Complete).
5. Copy all cells which has data include titles and paste it into A1 cell of a sheet whose name is "入力"("Input") in CalcMultiplyRes.xlsx.
6. Save 結果用紙("Result Paper") as a PDF. This is all process. Thank you.
